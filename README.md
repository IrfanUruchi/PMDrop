# PMDrop

PMDrop is a self-hosted local network sharing platform that allows users to transfer files, text, and clipboard content between devices through a web interface. The project is designed to be lightweight, private, and easy to deploy on a Raspberry Pi, NAS, mini PC, or any Linux server.

The primary goals of PMDrop are simplicity, cross-platform compatibility, and complete independence from third-party cloud services and user accounts.
